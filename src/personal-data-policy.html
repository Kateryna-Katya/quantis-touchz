Personal Data Processing Policy

1. General Provisions

This Personal Data Processing Policy (hereinafter — the "Policy") defines the procedure and conditions for processing personal data undertaken by the Quantis-Touchz platform team (hereinafter — the "Operator"), and establishes measures to ensure the security of this data in accordance with EU digital security regulations.

1.1. The Operator's most important goal is to respect human rights and freedoms during data processing, including the protection of privacy and personal secrets in the context of using AI technologies.

1.2. This Policy applies to all information that the Operator may obtain about visitors (hereinafter — "Users") of the quantis-touchz.org website.

2. Key Definitions

  • Website — the collection of graphic and information materials available at quantis-touchz.org.
  • User — any visitor to the platform.
  • Personal Data — any information relating to a specific User (name, email, phone).
  • Data Processing — any actions from collection to deletion of information performed to ensure the operation of Quantis-Touchz services.

3. Data We Process

The Operator may process the following categories of data:

  • Data Provided by You:
    • Name;
    • Email address;
    • Contact phone number.
  • Technical Data:
    • Anonymous visit data ("cookies"), IP address, and device type collected to optimize AI interface performance.

4. Purposes of Data Processing

  • Identifying the User to provide access to AI practices and guides.
  • Establishing feedback and consulting on the implementation of innovations.
  • Informing about new solutions via newsletters. The User can always opt-out by writing to contact@quantis-touchz.org.
  • Improving the website structure based on user experience.

5. Legal Grounds

The Operator processes data only with your voluntary consent, expressed through filling out forms on the website or through your browser settings (for cookies).

6. Security and Storage

  • We take technical measures to protect data from unauthorized access. Databases are protected according to French security standards.
  • Data is never transferred to third parties, except in cases provided by EU law.
  • The processing period is unlimited, but you can withdraw your consent at any time via email.

7. Final Provisions

  • The current version of the Policy is always available on this page. Any changes take effect from the moment of publication.
  • Direct questions and data deletion requests to: contact@quantis-touchz.org.
